How to Test the API

You can test the accounts API with 4 parameters, all of which are optional. Here are some examples of how to use these parameters:

  • Minimum Balance: /accounts/?min_balance=100
  • Maximum Balance: /accounts/?max_balance=5000
  • Consumer Name: /accounts/?consumer_name=Matthew Kao
  • Status: /accounts/?status=active
  • Combining Parameters: /accounts/?min_balance=100&max_balance=5000&consumer_name=Matthew Kao&status=active

The datastore used is the default SQLite database provided by Django.

To run tests, use the command python manage.py test in your terminal. The CSV data is automatically imported when the server starts, ensuring the database is ready for immediate testing.
